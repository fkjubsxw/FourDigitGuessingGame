# FourDigitGuessingGame 🎮
一个用 Python + Tkinter 做的 4 位数字猜谜小游戏，带图形界面，简单又好玩！

---

## 🎯 游戏规则
1.  系统会随机生成一个**不重复的 4 位数字**（比如 `2571`）。
2.  你在输入框中输入一个 4 位数字，点击「查询」。
3.  系统会提示你两个信息：
    - ✅ 猜对的数字且位置也对的个数
4.  游戏自带**草稿纸功能**：你输入过、查询过的数字，以及它们对应的正确个数，都可以记录在草稿纸上，方便你一步步推理答案。
5.  根据提示和草稿纸记录调整猜测，直到 4 个数字全部猜对，即可获胜。
6.  支持「重新开始」，随时开启新一局游戏。

---

## 🚀 两种使用方式
### 方式一：直接运行源码（适合有 Python 环境的用户）
1.  确保你的电脑安装了 Python 3.x。
2.  克隆或下载本仓库到本地。
3.  运行 `main.py` 即可启动游戏。

### 方式二：下载打包好的 exe（无需安装 Python）
- 直接去本仓库的 **Releases** 页面下载最新版本的压缩包，解压后双击 `main.exe` 就能玩。

## ⚠️ 注意事项
- 输入必须是**4位纯数字**，不能有字母、符号，也不能重复数字，否则会提示输入不合法。
- 打包好的 exe 文件在 Releases 中发布，源码和 exe 都是公开免费的。
